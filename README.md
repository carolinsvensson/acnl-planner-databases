This project is set up to manage the current databases used in the Planner for AC:NL app.

Right now I am in need of help with the interior translations.
I am currently working on the German one, so I have not added it here yet.
The parts that need to be translated are **name**, **clothing_style**, **obtained_from** and any text written in **sell_value** and **purchase_value**.

For example 
`"name": "block floor",` translates to
`"name": "Bruchsteinboden",`
and
`"purchase_value": "880 bells; 3 MEOW Coupons",` translates to
`"purchase_value": "880 Sternis; 3 Camping-Coupons",`

###### HOW DO I HELP!?
**Step 1**
* Start by creating a free GitHub account
* Download and install [GitHub Desktop](https://desktop.github.com/)
* Press the green button that says *Clone or download* and choose *Open in Desktop*
* Clone the repository. Now you have the files on your computer, time to edit! :D

I normally use [Notepad++](https://notepad-plus-plus.org/download/v7.6.3.html) to edit these files, but you can use any text editor you're comfortable with. When you're happy with the changes you've made, move on to step 2.

**Step 2**
* Open up the GitHub Desktop application again
* You are now able to see the changes you have made. If you want to, you can write a title and description of what you have changed (ex Added the missing item Vaulting Horse)
* Press the *Commit to develop* button
